# 家乡介绍 App - 需求文档

## 概述

一款展示中国城市风貌的 Flutter 移动端 App。首期实现太原城市页面，后续团队成员可各自添加自己的家乡城市。

## 核心功能

### 1. 城市页面（4 个 Tab）

| Tab | 功能 | 内容 |
|-----|------|------|
| **介绍** | 城市基础信息展示 | 人口/历史/面积统计、城市简介、标签（历史文化、地标等）、气候特征、历史沿革、面食文化、交通贴士 |
| **景点** | 景点浏览与详情 | 横向滑动卡片（6 个热门景点）、竖向列表（7 个其他景点）、点击弹出详情（描述+时间+地址+建议），触觉反馈 |
| **美食** | 美食浏览与详情 | 2 列网格（10 道美食）、热度评分（1-5 星动画）、点击弹出详情（描述+推荐店+人均），触觉反馈 |
| **路线** | 旅行路线规划 | 5 条主题路线（龙城印象/三晋文脉/太原味道/亲子探索/古建光影），步骤可展开，一键复制路线 |

### 2. Tab 交互

- 点击 Tab 切换
- **左右滑动**切换
- Tab 指示器实时跟随滑动
- 切换时内容淡入过渡
- 滚动时 Tab 栏出现阴影

### 3. Hero 头部

- 每个 Tab 对应不同背景色和图标，切换时平滑过渡
- 显示省份标签、城市名、副标题
- 右上角语言切换按钮

### 4. 多语言

- 支持中文 / English / 한국어 三语
- 点击 🌐 按钮弹出语言选择面板
- 即时切换，所有文本联动更新

### 5. 卡片视觉

- 卡片顶部渐变占位图（离线可用）
- 可替换为真实照片
